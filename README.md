# Flipper Zero Collection

Dieses Repository enthält eine Sammlung von Skripten, Grafiken und Payloads für den Flipper Zero.

Haftungsausschluss: Alle Inhalte dienen ausschließlich Bildungszwecken. Die Verwendung erfolgt auf eigene Gefahr. Benutzen Sie diese Tools nur an Systemen, für die Sie eine ausdrückliche Erlaubnis besitzen.

---

## Verzeichnisstruktur

### BadUSB
DuckyScript-Payloads optimiert für verschiedene Betriebssysteme.
* ASCII-Art: Darstellungen für Notepad (Batman, Anonymous, Skull).
* Automatisierung: Skripte für Systemkonfigurationen und Downloads.
* Layout-Hinweis: Die meisten Skripte sind für das de-DE Layout vorkonfiguriert.

### Sub-GHz
Dateien für Funkfrequenz-Analysen und Aufzeichnungen.

### Graphics
Angepasste Icons und Animationen für das Flipper-Display.

---

## Installation und Nutzung

1. Klonen Sie das Repository auf Ihren lokalen Computer:
   git clone https://github.com/DEIN_NUTZERNAME/flipper.git

2. Übertragung auf das Gerät:
   Verbinden Sie den Flipper Zero mit Ihrem PC und nutzen Sie die qFlipper App, um die Dateien in die entsprechenden Verzeichnisse auf der SD-Karte zu kopieren.

---

## Hardware und Software

* Firmware: Offical
* SD-Karte: 32GB
* Primäres Tastaturlayout: de-DE (Deutsch)
